# Gestión de Datos – SQL / T-SQL

Proyecto académico orientado al diseño e implementación de un Data Warehouse utilizando SQL Server.
Incluye modelado dimensional, tablas de hechos y dimensiones, consultas analíticas y procedimientos almacenados.

## Tecnologías
- SQL Server
- SQL / T-SQL
- Modelado de datos
- Data Warehouse

## Contenido del repositorio
- Scripts de creación de tablas
- Scripts BI (hechos y dimensiones)
- Consultas analíticas
- Diagramas (DER y modelo BI)

> Proyecto desarrollado con fines académicos.
